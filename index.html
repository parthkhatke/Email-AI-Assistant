About This Project

This Email AI Reply Assistant is built using Java SpringBoot, HTML, CSS, and JavaScript. It features a modern glassmorphism design with smooth animations and responsive layout. The application connects to an Google Gemini API using backend service to generate intelligent email responses based on your content and preferred tone. The backend is deployed on Render using Docker image
